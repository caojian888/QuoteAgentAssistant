# 铜编织线成本计算

通过分离以下内容来估算铜编织线零件成本：

1. 图纸上明确可见的内容
2. 属于成本计算模板的内容
3. 计算总价格

除非用户明确要求含税价格，否则首先提供估算的不含税总价。

## 工作规则

始终明确遵守这些规则：

- 默认最终输出为**不含税（未税）**
- 使用**图纸重量**作为定价重量基础，除非用户覆盖
- 将参考Excel仅视为**逻辑模板**，而非权威重量来源
- 如果结果依赖于推断，明确标记推断
- 不要假装推断的工艺规则是已确认的公式
## 所需输入

提取所见的图纸的以下信息，不得编造：

- 零件编号/图纸编号
- 材料等级
- 编织线规格
- 零件是两端、三端还是多端
- 总长度
- 编织段长度
- 连接区域数量
- 端子长度
- 宽度
- 厚度
- 孔数量
- 孔直径
- 是否有中间连接区域
- 是否有热缩管/绝缘/保护套管
- 图纸重量
- 电气连接区域或固化区域的任何特殊说明

如果关键输入缺失，继续进行初步估算并明确列出缺失项目。

## 核心工作流程

### 步骤1：计算前先阅读图纸

首先提取这些信息：

- 零件编号
- 材料/规格
- 图纸重量
- 总长度
- 宽度
- 厚度
- 端子数量
- 连接区域数量
- 孔型
- 热缩或绝缘说明
- 可见工艺说明

在确定零件是以下类型之前，不要开始成本计算：

- 两端编织连接器
- 三端编织连接器
- 多端编织连接器

### 步骤2：使用图纸重量作为成本基础

这是本技能的默认规则。

使用：

- 图纸重量作为净定价重量

如果模板需要毛重但未直接给出，保守地从以下内容估算：

- 图纸重量 + 小工艺余量

当前默认毛重估算规则：

- 毛重 = 图纸重量 + 0.005 kg

如果零件明显更大或更复杂，且此余量看起来太小，明确说明并使用保守的临时估算。

### 步骤3：应用通用材料成本模板

使用当前通用模板，除非用户覆盖。

#### 默认假设

如果未另行指定，使用：

- 铜价参考（含税）：80000 元/吨
- 铜线金额系数：(铜价 / 1000 + 20) 元/公斤
- 转换用增值税率：13%
- 利润率：5%
- 运输默认约为图纸重量（元/件）
- 冲孔从每个圆孔2元开始作为基本规则
- 小值间接费用项目在没有更好规则时可使用模板式分层值

#### 不含税转换

除非用户明确要求含税定价，否则将成本计算工作流程视为：

1. 以模板的工作惯例计算工艺和材料项目
2. 遵循采购式表格逻辑时，先计算含税总价
3. 通过除以1.13将最终结果转换为不含税

- 不含税总价 = 含税总价 / 1.13
- 含税总价 = 不含税总价 × 1.13

#### 材料成本

当前编织线采购模型的默认公式：

- 材料成本 = ((铜价 / 1000) + 20) × 图纸重量

其中：

- 铜价为含税，使用元/吨
- 图纸重量直接取自图纸
- `+20` 项是当前校准中学习到的模板系数

除非用户明确要求，否则不要用Excel重量替换图纸重量。

### 步骤4：应用工艺成本模板

对于当前编织线采购式模型，将工艺成本分为：

- 成型
- 冲孔
- 边缘打磨
- 镀锡/热缩管
- 废料损失
- 管理费
- 包装
- 运输
- 折旧

#### 成型

成型尚未有确认的公式。

主要从以下方面推断：

1. 连接区域数量
2. 柔性段数量
3. 总长度
4. 整体结构复杂性

当前工作规则：

- 使用分层估算，而非假装存在确认的公式
- 简单小零件可能在4-5左右
- 中等多区域零件可能在30-35左右
- 大型重型零件可能在270以上

估算时，说明哪些结构特征驱动了成型费用。

#### 冲孔

当前基本规则：

- 圆孔冲孔从每个孔2元开始

仅在明显需要时向上调整：

- 大直径
- 混合孔尺寸
- 特殊槽/异形孔
- 重型截面难度

如果图纸明确显示4个标准圆孔，一个好的初步估算是8元。

#### 边缘打磨

使用分层规则：

- 无或可忽略 -> 0
- 普通零件 -> 默认2元（无特殊要求时按此执行）
- 更大/更多接触边缘工作 -> 5

#### 镀锡/热缩管

基于是否需要绝缘或热缩以及柔性段覆盖程度使用分层规则：

- 无或最小 -> 0 至 3
- 小零件/短覆盖 -> 3 至 4
- 正常多段覆盖 -> 20
- 大零件/大规格/长覆盖 -> 40

#### 废料损失

当没有更明确的规则时，使用小分层估算：

- 小零件 -> 0.1 至 0.2
- 中等零件 -> 0.5
- 大零件 -> 2

#### 管理费

默认占位符：

- 管理费 = 0.1

#### 包装

使用小分层估算：

- 小零件 -> 0.1 至 0.2
- 大零件 -> 0.5

#### 运输

当前推断规则：

- 运输 ≈ 图纸重量

必要时保持实用的四舍五入值。

#### 折旧

在当前示例中通常很小。

- 除非用户给出更明确的方法，否则视为接近零

### 步骤5：计算利润前的总计

使用：

- 小计 = 材料成本 + 成型 + 冲孔 + 边缘打磨 + 镀锡/热缩管 + 废料损失 + 管理费 + 包装 + 运输 + 折旧

### 步骤6：计算利润

使用：

- 利润 = 小计 × 5%

### 步骤7：计算最终价格

使用：

- 含税总价 = 小计 + 利润
- 不含税总价 = 含税总价 / 1.13

## 输出格式

每个响应应包含以下用plain language编写的部分：

### 1. 图纸识别

列出：

- 零件编号
- 材料/规格
- 端子数量
- 总长度
- 宽度/厚度
- 孔型
- 是否有中间连接区域
- 图纸重量
- 热缩/绝缘说明

### 2. 成本计算基础

明确说明：

- 使用图纸重量作为定价基础
- 使用哪些默认假设
- 输出是不含税还是含税
- 哪些工艺项目直接从采购式模板中学习
- 哪些工艺项目通过分层规则推断

### 3. 成本分解

列出：

- 材料成本
- 成型
- 冲孔
- 边缘打磨
- 镀锡/热缩管
- 废料损失
- 管理费
- 包装
- 运输
- 折旧
- 利润
- 总估算成本（不含税）
- 总估算成本（含税，仅在有用或请求时）

### 4. 风险/待确认项目

列出所有不确定性，例如：

- 图纸尺寸不明确
- 连接区域数量可能误读
- 总长度可能误读
- 成型规则是推断的，而非确认的
- 冲孔规则可能需要为特殊孔调整
- 热缩/绝缘处理可能影响实际车间成本
- 小值间接费用项目仍然是模板式估算

## 强制自我检查

在最终确定之前，检查所有这些：

- 我是否正确识别了连接区域结构？
- 我是否使用图纸重量，而非模板重量？
- 我是否应用了当前模型中的铜线金额公式？
- 我是否根据孔数量和复杂性估算了冲孔？
- 我是否将成型视为推断的结构成本项目，而非虚假的精确公式？
- 我是否将最终结果默认为不含税定价？

如果任何答案不确定，明确提及。

- 参考Excel用于学习逻辑，而非覆盖图纸重量

- 运输当前最好近似为图纸重量（元/件）

- 默认最终输出为不含税定价（未税）
