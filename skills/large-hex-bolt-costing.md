# 外六角大螺栓成本计算
通过首先从图纸中提取几何规格、材质与工艺要求，然后将它们映射到结构化的成本计算模型中来计算螺栓零件成本。

## 工作规则
始终分离这三个层次：
1.  图纸上明确显示的内容（必须100%提取，不得编造）
2.  从标准成本模板中推断的内容（如工序单价、废料率、费率）
3.  最终计算得出的总价格与成本结构占比

## 所需输入
从图纸中提取以下信息，仅以图纸标注为依据：
-  成品名称/品名（如“外六角螺栓M48×260 10.9级”）
-  云星空代码/ERP号/料号
-  规格参数：公称直径d、公称长度L、螺纹规格、性能等级（如10.9级）
-  材质（如合金钢/碳钢，图纸未标注时按性能等级推断）
-  图纸标注重量（如4.8kg/件）
-  核心几何尺寸：杆部直径、头部对边尺寸、螺纹长度
-  工艺与技术要求：
   -  热处理要求（调质处理等）
   -  螺纹加工方式（热处理后滚压螺纹）
   -  表面处理要求（参照标准如EN ISO 10683）
   -  外形标准（如ISO 4014）
-  单套用量（若图纸关联BOM，需提取，无则默认1件）

## 识别方式
**预处理要求：**
接收到图纸后，**第一步必须先用 trace-to-svg 技能将 PDF/图片转换为 SVG 矢量图**，再基于矢量图交给 Seed 2.0 视觉模型识别，矢量图识别精度更高，能大幅提升直径、长度标注及技术要求文字的准确性。
**必须使用 Seed 2.0 视觉模型直接识别图纸图片/PDF，不得使用 OCR。** 视觉模型对图纸几何形状、尺寸标注、技术要求文字的识别精度更高。

## 核心工作流程
### 步骤1：图纸信息提取与交叉校验
首先从图纸中提取并确认以下关键信息：
-  公称直径d（如M48）、公称长度L（如260mm）
-  性能等级（如10.9级，推断材质为中碳合金钢）
-  图纸标注重量（如4.8kg/件）
-  技术要求中的热处理、螺纹加工、表面处理要求
-  料号/规格号/图号
在确认零件为外六角螺栓后，进入下一步成本计算。

### 步骤2：重量基准确认与理论重量交叉验证
这是成本计算的核心基础，优先采用以下顺序确定重量：
1.  图纸标注的成品重量（默认优先使用）
2.  几何推导的理论重量（用于交叉校验）

#### 理论重量计算公式
外六角螺栓理论重量 = 头部重量 + 杆部重量
-  头部重量（六角头）：
  `V_head = (3√3/2) × s² × h`
  其中：s为头部对边尺寸，h为头部高度；合金钢密度ρ=7.85g/cm³，`m_head = V_head × ρ`
-  杆部重量（含螺纹段）：
  `V_shank = π × (d/2)² × L`
  其中：d为公称直径，L为杆部有效长度；`m_shank = V_shank × ρ`
-  理论总重量 `m_theory = m_head + m_shank`

#### 重量校验规则
-  若理论重量与图纸标注重量偏差≤±5%，直接采用图纸标注重量
-  若偏差>±5%，以理论重量为准，并在报告中标注“图纸重量偏差较大，采用理论重量计算”

### 步骤3：按标准成本模板分项计算
#### 成本项构成（与Excel模板完全对应）
| 成本项 | 计算逻辑 | 说明 |
|--------|----------|------|
| 原材料成本 | `(理论重量 × (1 + 废料率)) × 材料单价` | 废料率默认5%（下料损耗），材料单价按对应材质市场价（如10.9级合金钢按对应含税价） |
| 下料费 | 按件计价，单价与公称直径挂钩（如M48规格固定单价） | 棒料切割工序费用 |
| 车削加工费 | 按件计价，单价与规格挂钩 | 加工六角头外形、杆部倒角 |
| 滚压螺纹费 | 按件计价，单价与螺纹规格挂钩 | 热处理后滚压螺纹工序费用 |
| 热处理费 | `图纸重量 × 热处理单价（元/kg）` | 调质处理等工艺费用 |
| 表面处理费 | `图纸重量 × 表面处理单价（元/kg）` | 按图纸要求的工艺（如达克罗、热镀锌）计价 |
| 加工费合计 | 下料费+车削费+滚压螺纹费+热处理费+表面处理费 | 所有工序费用总和 |
| 管理费 | `(原材料成本 + 加工费合计) × 3%` | 固定费率，与绝缘纸模板一致 |
| 运输+包装费 | `图纸重量 × 3元/kg` | 按成品重量计算，与绝缘纸模板一致 |
| 利润 | `(原材料成本 + 加工费合计 + 管理费 + 运输包装费) × 5%` | 固定利润率，与绝缘纸模板一致 |
| Should cost | 上述所有成本项之和 | 目标成本 |
| 原材料占比 | `原材料成本 / Should cost × 100%` | 保留整数 |
| 加工费占比 | `加工费合计 / Should cost × 100%` | 保留整数 |
| 利管运占比 | `(管理费 + 运输包装费 + 利润) / Should cost × 100%` | 保留整数 |

### 步骤5：自动校验与异常处理
#### 自动校验规则
1.  几何校验：理论重量与图纸标注重量偏差≤±5%，否则触发报警
2.  成本结构校验：原材料占比应在30%-50%之间（大螺栓为加工密集型零件，低于绝缘纸的75%-90%），否则提示“成本结构异常”
3.  工艺完整性校验：必须包含热处理、螺纹加工、表面处理三项核心工艺，否则提示“图纸工艺要求不全，成本估算存在偏差”

#### 异常处理规则
1.  图纸参数缺失：自动填充默认值并在报告中标红提示（如无材质标注，按10.9级推断为合金钢；无表面处理要求，默认按基础镀锌计价）
2.  规格识别失败：立即终止计算，输出“图纸规格参数无法识别，请检查直径/长度标注”
3.  成本结构异常：自动标注“原材料占比异常，建议复核材料单价或工序成本”
