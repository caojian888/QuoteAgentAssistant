# 图纸品类识别与路由

接收到产品图纸（PDF / 图片）后，识别图中物料品类，并将请求派发到对应的成本计算技能。本技能不做成本计算，只负责「识别 → 派发」。

## 工作规则

1. 使用当前 Codex 环境可用的图纸/PDF/图片理解能力识别图纸；不要要求调用某个固定外部视觉模型，除非当前环境明确提供该模型。
2. 优先从图纸的"品名 / 名称 / Description"字段提取品类关键词，位置通常在标题栏、明细表第一行、或顶部技术说明区
3. 识别失败、或品类不在路由表中时，**立即返回错误**，不要强行猜一个最近的品类去派发

## 工作流程

### 步骤 1：图纸识别

从图纸中提取以下信息：
- **品名**（核心，用于路由判断）
- **图号 / 物料号**（透传给下游 skill）
- **关键尺寸**（透传给下游 skill）

### 步骤 2：品类匹配

按下方"路由映射表"匹配品名关键词，确定目标技能。匹配原则：
- 关键词出现在品名中即命中（不要求完全等于）
- 同一行内任一关键词命中即可
- 多行同时命中（极少见）：以"主体零件"优先，辅料不参与判断

### 步骤 3：派发调用

调用匹配到的成本计算技能，**原始图纸作为输入透传**。本技能不做任何尺寸提取或成本预估，全部交给下游 skill。

## 路由映射表

| # | 识别到的品类关键词 | 派发到的技能 |
|---|---|---|
| 1 | 铜母排 / 铜排 / 铜件 / 铝母排 / 铝排 / 铝件 | `copper-busbar-costing` |
| 2 | 铜编织线 / 铜编织带 / 软连接 / 柔性铜连接件 | `copper-braided-wire-costing` |
| 3 | 绝缘纸 | `insulation-paper-costing` |
| 4 | 大六角螺栓 | `large-hex-bolt-costing` |
| 5 | 钣金件（PLATE STEEL BEND） | `sheet-metal-part-costing` |

## 异常处理

| 异常情况 | 处理方式 |
|---|---|
| 视觉模型未能识别出品名 | 返回："未能从图纸识别品名，请提供更清晰的图纸或人工指定品类" |
| 识别出的品名不在路由表中 | 返回："品类「X」暂不支持成本计算。当前已支持品类：铜排 / 编织线 / 绝缘纸 / 大六角螺栓 / 钣金件" |
| 同一张图纸出现多个品类关键词 | 以**主体零件**优先（明细表第一行 / 标题栏品名），辅料 / 配件不参与判断 |

## 示例

### 示例 1：铜母排图纸

**输入**：一张 TMY-100×10 铜母排图纸 PDF

1. 调用 doubao-seed-2.0-pro 识别品名 → "TMY-100×10 铜排"
2. 匹配关键词"铜排" → 命中第 1 行
3. 调用 `copper-busbar-costing`，输入原始图纸

### 示例 2：绝缘纸图纸

**输入**：一张 0.5mm DMD 绝缘纸图纸

1. 识别品名 → "0.5mm DMD 绝缘纸"
2. 匹配关键词"绝缘纸" → 命中第 3 行
3. 调用 `insulation-paper-costing`

### 示例 3：未支持品类

**输入**：一张 PCB 板图纸

1. 识别品名 → "FR4 双面 PCB"
2. 路由表中无匹配项
3. 返回："品类「PCB」暂不支持成本计算。当前已支持品类：铜排 / 编织线 / 绝缘纸 / 大六角螺栓 / 钣金件"
